Setup

Unlike other frameworks, strap.less is not and will never be a complete, end-to-end style solution. The core principals driving its design are simplicity, modularity and extensibility. Take only what you need and modify it to your purposes, while leaving the foundation intact to allow for easy upgrades.

To start with, your project should look something like this:


less/
├─┬ bones/ # leave these alone
│ ├── _config.less
│ ├── _mixins.less
│ └── ...
├─┬ skin/ # customize these after @import ...
│ ├── _config.less
│ ├── _mixins.less
│ └── ...
├─┬ page-specific/
│ │   # page-specific, non-universal styles should be separate from the main project
│ │   # @import _config.less & _mixins.less
│ │   # wherever shared variables or mixins are needed
│ ├── page-specific-styles.less
│ ├── page-specific-responsive.less
│ └── ...
├── strap.less # all universal modules @imported here from skin/
└── page-specific.less # modules from page-specific dir @imported here
